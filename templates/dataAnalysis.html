Data Analysis

All compilation of data was done in python. All analysis, model fitting, and parameter extraction was done in python using PYROOT.

Results of Electronic Chain Calibration

Because a test pulse of a sharp rise and linear decay was chosen, this introduced a trend within the analysis. See figure (\ref{PM_amplitude_change}) and (\ref{PM_frequency_change}) to see this negative trend. This is a result by changing the slope of the decay of the test pulse. This is done by changing the settings on the test pulse. To find the appropriate gain from each data set, a linear line was fitted to the trend and the intercept was extracted. This intercept is what was averaged between the two data sets for each PM.

Results of PMT Calibration

It should be noted that the weighted averaged $Q_1$ for PMT0 does not include waveform 5. This is because waveform 5 does not have enough non-zero PE traces, allowing for a reliable fit for $Q_1$. Thus it was not included.
